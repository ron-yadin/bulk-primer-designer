Bulk Primer Designer Flask App

Description

  • This is a flask webapp through which a user can upload a csv of amplicon sequences, and then download a zip file containing both their input csv and output csv with optimal primer selections.
  • All submission events, inputs, and primer options + scores are stored in a MySQL database available at localhost:8080
  • The backend checks assumptions - like the expected file extension & columns, and displays helpful error messages when not met.

Instructions

  • Upload an input csv file with two columns: "amplicon name" and "sequence"
  • Returns a downloadable zip file containing input csv & output csv, with primer options + scores and metadata
